# Bethini Pie: INI editor designed to allow advanced customization of game configuration settings.

**Bethini Pie** is a universal tool designed to make editing **INI configuration files** simple via an intuitive **graphical user interface**. It allows you to tweak game settings, optimize performance, and personalize configurations without manually opening INI files.

## 🧰 Dependencies

- **.NET 7 Runtime** installed (if not already)  
- The target game installed on your PC  
- Optional: **Backup copies** of your INI files

---

## 🚀 Usage Guide

### Step 1: Launch BethiniPie
- Open the application; no installation is required (portable).

### Step 2: Select Game or INI File
- Either pick a supported game from the list or browse to any INI file manually.

### Step 3: Modify Settings
- Use the graphical sliders, checkboxes, and dropdowns to tweak values.  
- Changes are automatically highlighted; you don’t need to manually open the INI file.

### Step 4: Apply Changes
- Click **Save** to apply your modifications to the selected INI file.  
- Always make a backup when prompted.

### Step 5: Launch Game
- Run your game to see the performance or gameplay changes in effect.

## 🔧 Troubleshooting

| Problem                         | Solution                                                                 |
|---------------------------------|--------------------------------------------------------------------------|
| App doesn’t open                 | Ensure .NET 7 Runtime is installed and restart the app.                  |
| Changes not applied              | Check file permissions; run BethiniPie as Administrator.                 |
| INI file not recognized           | Ensure it is a standard INI format; try opening via the manual browse option. |
| Game crashes after edits          | Restore the backup INI and test changes incrementally.                   |

## 🧠 Tips & Notes

- Always **back up INI files** before editing.  
- BethiniPie supports **portable usage**—no installation required.  
- Use sliders and presets for optimal performance tuning.  
- For advanced users: manually edit values directly in the interface for custom tweaks.  
- Works with **official game builds**; modded INIs may behave unpredictably.
